# vue

## Requirements

- Git
- Docker
- Docker Compose
- GNU/Make

## Repository Installation

```console
$ git clone git@github.com:QALT/vue.git ~/github.com/QALT/vue
$ cd ~/github.com/QALT/vue
```

## Dependencies Installation

```console
$ make install
```

## Development

```console
$ make development
```

## Production

```console
$ make production
```
